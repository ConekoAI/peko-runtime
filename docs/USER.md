## Project Vision

Pekobot is designed to be:
- **Lightweight** — Small binary, fast startup
- **Standalone** — Works without Coneko for local use
- **Embeddable** — Easy to integrate into other systems
- **Multi-agent** — Supports agent-to-agent communication
- **Secure** — Uses ed25519 identity and DID-based addressing
